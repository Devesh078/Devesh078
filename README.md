## About Me

I'm a Computer Engineering student at **Xavier Institute of Engineering, Mumbai**, focused on building production-grade **full-stack** and **AI-integrated** systems end to end — from database schema to deployed API to native mobile UI.

My core strength is **backend architecture**: designing REST APIs, authentication systems, and data models that scale cleanly, then layering **AI/ML capability** on top — from intent classification to Retrieval-Augmented Generation (RAG) pipelines using LLMs like Google Gemini. I care about systems that are not just functional, but *engineered* — secure, maintainable, and built with a product mindset rather than a tutorial mindset.

I'm currently deep in a fitness-tech platform combining nutrition science, recovery analytics, and conversational AI coaching, while sharpening my computer science fundamentals through rigorous DSA and systems-level C++ work.
